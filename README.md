# ESP-Doorbell
ESP-Doorbell is a decentralized smart notification system built using dual ESP32 microcontrollers, integrated via the ESP-NOW protocol for low-latency wireless communication. This project features a remote trigger node (Pushbutton) and a central receiver hub that utilizes a 16x2 I2C LCD for visual status updates and a PWM-driven buzzer for audible alerts, all developed and simulated within the PlatformIO and Wokwi ecosystems.
